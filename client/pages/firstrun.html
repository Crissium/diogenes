First Run Setup (you're almost there...)

Diogenes needs to know where on your computer the databases of ancient texts are located. If you need to change the locations later, you can do this via the application menu: File -> Diogenes Settings.

The databases are not distributed with Diogenes; they must be acquired separately.

You will need at least one database to use Diogenes.
